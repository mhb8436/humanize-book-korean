## 도입부 등장 패턴

본 책의 모든 장은 김과장의 시점에서 시작합니다. 도입부 표준 패턴:

```
김과장이 [사내 시스템에서 받은 / 발주처에서 들어온 / 책상에 도착한]
[RFP / 신규 사업 소스 코드 / 보안성 검토 요청서]를 ...
```

**금지 표현**:
- "김주임" — 10년 차에 주임은 부자연스러움 (사용자가 직접 김과장으로 변경)
- "RFP가 떨어진다" → "RFP가 들어온다 / 도착한다"
- "김과장의 머릿속에 자동으로 매핑된다" → "김과장이 곧바로 ~를 떠올린다"

## 페르소나 보존 규칙

humanize 윤문 시 다음을 절대 변경하지 않습니다:

- 이름 "김과장"
