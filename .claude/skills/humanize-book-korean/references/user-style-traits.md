# 한국 IT 책의 톤 특징 (Style Traits)

본 스킬은 한국에서 출간된 IT 기술서의 자연스러운 톤을 기준으로 윤문합니다. 한국 IT 기술서 19권 분석(사용자 자바책 등) 결과 다음 8가지 특징이 공통됩니다.

## 1. 사실로 시작합니다

도입부와 절 시작이 **정의·사실 진술**입니다. 자기 정의("이 책의 자리"), 환영 멘트("함께 시작합시다"), 약속("이 책의 약속")으로 시작하지 않습니다.

```
좋음:  "자바는 객체지향 언어이다."
       "이번 장에서는 클래스의 자세한 내용을 배운다."
나쁨:  "이 장은 객체지향이라는 매혹적인 세계로 여러분을 안내합니다."
       "이 책이 자리 잡는 곳은 두 흐름의 교차점입니다."
```

## 2. 비유 명사 0건

한국 IT 책에는 다음 비유 명사가 **거의 등장하지 않습니다**:

- "~의 자리", "~의 흐름", "~의 교차점"
- "~의 토대", "~의 골격", "~의 표면"
- "~의 정체성", "~의 약속", "~의 본질"
- "~의 시드(seed)", "~의 디딤돌"

영어 IT 책의 "the foundation/framework/surface of X" 직역이 한국 IT 책에는 안 어울립니다.

## 3. 환영·약속·결의 멘트 0건

미국 IT 책의 패턴 ("Let's begin together", "What this chapter promises")이 한국 IT 책에는 없습니다.

```
나쁨:  "함께 시작해 보시죠"
       "이 장에서 만드는 것은 코드가 아니라 ~의 골격입니다"
       "다음 페이지에서 만난다"
```

## 4. 자기 마케팅·자기 평가 0건

한국·일본 IT 책은 본문 안에서 자기 책을 평가하지 않습니다. 미국 책의 "Why this chapter matters" / "★ Key chapter" 패턴은 안 씁니다.

```
나쁨:  "★ 핵심 차별화 장"
       "이 책이 시중의 다른 ~ 서적과 가장 크게 다른 부분입니다"
       "이 장의 가장 큰 차별화 가치"
```

마케팅·홍보는 표지·서평·출판사 보도자료의 영역입니다.

## 5. 메타·요약체 0건

한국 IT 책은 본문 안에서 다음 메타 표현을 안 씁니다:

- "한 줄 결론", "한 문장으로 정리하면", "한 줄로 줄이면"
- "다른 말로 표현하면 —", "한 마디로"
- "결론부터 말하면", "본질적으로"

이런 표현은 발표 슬라이드·블로그 포스트의 정리 패턴이지 책 본문 흐름이 아닙니다.

## 6. 연결어가 단순합니다

한국 IT 책의 연결어 빈도:

| 자주 씀 | 거의 안 씀 |
|------|------|
| 즉, 이는, 이때, 다음의 | 다른 말로 표현하면 |
| 또한, 물론, 그러나 | 결론적으로 |
| 그렇다면, 한편 | 시사하는 바가 |
| 다음과 같다, 위와 같이 | 주목할 만한 점은 |

## 7. 종결어미 (책별 선택)

- **격식 ~다체**: 학술서·전문 IT 서적 일부 (예: 사용자 자바책 2014)
- **격식 ~습니다체** (★ 권장): 한국 IT 책의 사실상 표준. 친근하면서 격식 유지

본 스킬은 ~습니다체를 기본으로 합니다. ~다체 입력은 ~습니다체로 일관 변환.

## 8. 문장은 길어도 사실로만 채웁니다

한국 IT 책은 한 문장에 정보를 많이 담지만 **비유·강조·감상이 없습니다**. 사실만으로 채웁니다.

```
좋음:  "자바는 1995년에 썬마이크로 시스템즈(지금은 오라클에 인수됨)의
       제임스 고슬링에 의해 만들어졌다. 객체 지향 프로그래밍 언어라고
       불리고 있고 오라클에서 무료로 제공하고 있다."

나쁨:  "자바라는 언어는 1995년에 태어났고, 그 탄생은 단순한 사건이 아니라
       프로그래밍 패러다임의 결정적인 전환점이 되었습니다.
       제임스 고슬링이라는 천재 개발자의 손에서 만들어진 이 언어는 ..."
```

## 자기 진단 체크리스트

본 스킬이 윤문한 결과를 사용자가 확인할 때 다음 8가지를 체크합니다.

- [ ] 사실로 시작했는가? (자기 정의·환영 멘트 0건)
- [ ] 비유 명사 0건? (자리/흐름/토대/골격/표면/정체성/약속)
- [ ] 환영·결의 멘트 0건? (함께/만나자/시작해 보시죠)
- [ ] 자기 마케팅 0건? (★/차별화/시중의 다른)
- [ ] 메타·요약체 0건? (한 줄 결론/다른 말로 표현하면)
- [ ] 연결어가 단순한가? ("즉/이때/또한/물론" 위주)
- [ ] 종결어미 일관? (~습니다체)
- [ ] 사실 중심? (비유·강조·감상 없음)

8가지 모두 통과해야 등급 A.
