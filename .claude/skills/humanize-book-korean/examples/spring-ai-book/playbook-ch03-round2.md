## 신규 발견 패턴 (v0.5.0 누적 후보)

### X-18 비유 동사

영어 "warps / bends / scatter / justifies" 직역:
- "응답을 크게 휩니다" → "응답을 크게 바꿉니다"
- "분포를 강하게 휩니다" → "분포를 강하게 바꿉니다"
- "여러 곳에 흩뿌리면" → "여러 곳에 나누어 두면"
- "이 차이가 이 장의 모든 코드를 정당화합니다" → "이 차이가 이 장의 모든 코드의 근거입니다"

### X-19 추가 비유 명사 (본 책 60+건 만연)

| 영어 비유 | 본 책 빈도 |
|------|------|
| 자산 (asset) | **54건** |
| 곡선 (curve) | 7건 |
| 토대 (foundation) | 5건 |

대안:
- "프롬프트가 자산임을" → "프롬프트를 코드 자원으로"
- "학습 곡선" → "학습 흐름 / 학습 과정" (단, "학습 곡선"이 굳어진 표현이면 보존)
- "이 토대를" → "이 정의를"

### X-20 공저 노트 조사 오타

- "이 장는" → "이 장은"
- "이 장와" → "이 장과"

자동 변환 sed 한 줄로 처리 가능.

## 잔류 의심 영역 (보존 결정)

- **헤딩 em-dash 27건** — 사용자 자바책 reference의 헤딩 패턴(`11장. 제네릭(Generic)과 어노테이션`)과 일치하므로 보존. v0.5에서 헤딩 em-dash를 별도 카테고리로 분리할지 검토 권장.
